# 答案方法
## 左右乘积列表
# 将索引左侧的乘以右侧的所有乘积,用两个数组L[i]记录i左边所有数字的乘积， `L[0]=1; L[i]=L[i-1]*num[i-1];`用R[i]记录i右边所有元素的乘积，`R[length-1]=1;R[i]=nums[i+1]*R[i+1];`最终 `result[i]=L[i]*R[i];`
## 基于输出数组
